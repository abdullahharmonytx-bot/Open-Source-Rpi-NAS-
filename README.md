# ALPHA

ALPHA is an open-source Raspberry Pi platform combining:

- NAS
- Local AI
- Smart Home
- Camera Monitoring
- ESP32 Integration
- Automation
- Touchscreen Panels

## Vision

ALPHA aims to be a self-hosted operating environment for Raspberry Pi.
